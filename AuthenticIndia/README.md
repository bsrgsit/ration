# Authentic India - Spice Store Mockup

A premium, interactive mockup for an e-commerce platform selling authentic regional Indian products.

## Features
- **Interactive India Map**: Hover over states to see their specialties.
- **Authenticity Details**: Click on a state (e.g., Andhra Pradesh, Kerala) to see famous products like Guntur Mirchi or Cardamom.
- **View Toggle**: Switch between the Map view and a Category Grid view.
- **Premium UI**: Dark theme with gold accents and glassmorphism effects.

## How to Run
Due to browser security restrictions (CORS) related to loading the SVG map file, you cannot simply double-click `index.html`. You must run it via a local server.

### Option 1: VS Code Live Server (Recommended)
1.  Open this folder in VS Code.
2.  Install the "Live Server" extension.
3.  Right-click `index.html` and select "Open with Live Server".

### Option 2: Python Simple HTTP Server
1.  Open a terminal in this directory.
2.  Run: `python -m http.server`
3.  Open `http://localhost:8000` in your browser.

## GitHub Hosting
This project is ready for GitHub Pages.
1.  Push this directory to your repository.
2.  Enable GitHub Pages in the repository settings (Source: `main` branch, `/` root or `/AuthenticIndia` folder).
